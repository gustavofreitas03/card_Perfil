Atividade Prática: Card de Perfil Interativo
Disciplina: Programação Web II
Alunos: Gustavo Freitas, Francisco vitor.

Atividade:
Criar uma estrutura contendo vários cards de perfis para diferentes usuários.
o Um card deve conter:
o Foto/avatar do perfil
o Nome do usuário
o Cargo/profissão
o Breve descrição/biografia
o Lista de habilidades (pelo menos 3)
o Contador de seguidores/seguindo
• Criar três botões para interação:
o Botão para alternar entre modo claro/escuro
o Botão para mudar a cor de fundo do card
o Botão para seguir/deixar de seguir cada perfil
• Estilizar o card para ficar visualmente atraente
• Centralizar o card na página quando clicar sobre ele
• Adicionar efeitos de hover nos botões
• Tornar responsivo
• Podem usar Flexbox ou Grid para organização
• Implementar as seguintes interatividades:
1. Alternar Modo Claro/Escuro: Mudar o tema da página inteira
2. Mudar Cor do Card: Alternar entre 3 cores pré-definidas para o fundo do card
3. Seguir/Deixar de Seguir:
o Alternar texto do botão entre "Seguir" e "Deixar de Seguir"
o Atualizar contador de seguidores (+1/-1)
o Mudar cor do botão (ex: azul quando segue, cinza quando não)

• Implementar troca de avatar
